# Лабораторный блок питания (STM32 / Embedded)

[![Видео-демонстрация]](https://youtube.com/playlist?list=PL1Fz6-nHFQaS-iLbRM0mhvYPwvbIchvey&si=EExdttfEdGngbPXH)

## О проекте

Этот проект представляет собой реализацию программируемого лабораторного блока питания на базе микроконтроллера STM32.

Устройство позволяет:

- Устанавливать выходное напряжение (вольты)
- Устанавливать выходной ток (амперы)
- Отображать текущие значения на встроенном дисплее
- Управление через [интерфейс: кнопки / энкодер]

Проект был создан в рамках стажировки в EPOS и опубликован как пример практического применения C и STM32 в задачах управления питанием.

## Особенности

- 🎛️ Интерфейс управления: [UART / GPIO / SPI / кнопки]
- 🛠️ Программная калибровка значений
- [📺 Встроенный дисплей / индикация]

## Стек технологий

- **Микроконтроллер:** STM32F407 
- **Инструменты:** [CubeMX / Makefile / STM32CubeIDE / Vim]
- **Язык:** C (HAL)
- **Отладка:** UART, ST-Link, Осцилограф
- **OS:** Bare-metal (без ОС)
